Ordino

For my Capstone project, my group and I developed a hardware inventory system for the IT department. This inventory system is focused on one room, 2208, in the Information Technology building. The system includes all the hardware in room 2208 such as TVs, monitors, workstations, routers and much more. Students will be able to see the status, description, type and location of items. In this case, a checkout process will be used to checkout and return hardware items. We called this system Ordino, which is latin for "order".
